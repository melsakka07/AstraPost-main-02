# 7 Thread Structures That Go Viral in MENA

"Sard" (سرد) — the Arabic term for threading — is a cornerstone of content culture on X in the Middle East. A well-crafted thread can generate thousands of impressions, hundreds of bookmarks, and meaningful engagement.

But not all threads are created equal. Here are 7 proven structures that consistently go viral in the MENA region.

---

## 1. The Historical Deep Dive

Middle Eastern audiences have a deep appreciation for history, especially forgotten stories and overlooked figures.

### Structure

| Tweet   | Content                   | Purpose          |
| ------- | ------------------------- | ---------------- |
| **1**   | Hook + Eye-catching image | Grab attention   |
| **2**   | Background context        | Set the stage    |
| **3-8** | The story (unfolding)     | Deliver value    |
| **9**   | Historical significance   | Add depth        |
| **10**  | CTA + Related topic       | Drive engagement |

### Example Hook

```
"Everyone knows the famous Arab scientists. But nobody talks about Al-Khwarizmi's forgotten rival who changed mathematics forever."

🧵 A thread...
```

### Why It Works

- Saudis and Emiratis love their history
- Educational content gets heavily bookmarked
- People share to show off their knowledge
- High replay value (people read multiple times)

## 2. The Local Startup Case Study

The Gulf region has produced incredible startups. People love breakdowns of how they did it.

### Structure

| Tweet | Content                 | Purpose             |
| ----- | ----------------------- | ------------------- |
| **1** | Revenue/Growth hook     | Create curiosity    |
| **2** | The problem they solved | Build context       |
| **3** | The solution            | Show innovation     |
| **4** | Growth strategy         | Reveal tactics      |
| **5** | Key lesson              | Make it actionable  |
| **6** | How to apply it         | Give value          |
| **7** | Recommended tools       | Help them implement |

### Example Hook

```
"Jahez went from a small delivery app to a $2B Saudi unicorn.

Here's exactly how they did it (and what you can copy):"
```

### Why It Works

- Entrepreneurship culture is booming in KSA/UAE
- People want to learn from local successes
- Numbers and specifics build credibility
- Highly shareable among business circles

## Anatomy of a Viral Thread Hook

Your first tweet determines whether anyone reads the rest. Here's what makes hooks work:

### Hook Formula

```
[Specific Result] + [Timeframe] + [Tension or Surprise] + [Thread Indicator]
```

### Examples

| Hook Type      | Example                                               |
| -------------- | ----------------------------------------------------- |
| **Numbers**    | "How this Saudi founder made $1M in 12 months"        |
| **Contrarian** | "Why most X advice is wrong for Arab audiences"       |
| **Listicle**   | "7 mistakes killing your engagement (and how to fix)" |
| **Story**      | "I got fired. Then I built a $50K/month business"     |

### Hook Best Practices

- ✅ Be specific (numbers, names, timelines)
- ✅ Create curiosity gap
- ✅ Make a bold promise
- ✅ Use "thread" indicator (🧵)
- ❌ Don't be vague
- ❌ Don't clickbait without delivering

---

## Thread Length Guidelines

| Topic          | Recommended Length | Why                    |
| -------------- | ------------------ | ---------------------- |
| **Quick tips** | 3-5 tweets         | Easy to consume        |
| **Tutorials**  | 7-10 tweets        | Cover steps properly   |
| **Stories**    | 10-15 tweets       | Full narrative arc     |
| **Deep dives** | 15-20 tweets       | Comprehensive coverage |

**Remember:** Quality > quantity. Every tweet should add value. Don't stretch to hit a number.

## Thread Promotion Strategy

Creating a great thread is only half the battle. Here's how to maximize reach:

### Pre-Launch

- Build anticipation with a teaser tweet
- Tag relevant accounts (with permission)
- Prepare thread graphics

### Launch

- Post at peak times (9 PM - 1 AM KSA)
- Use relevant hashtags
- Reply to initial comments immediately

### Post-Launch

- Quote tweet your own thread with key insights
- Turn into a blog post or LinkedIn article
- Create a graphics carousel version for Instagram
