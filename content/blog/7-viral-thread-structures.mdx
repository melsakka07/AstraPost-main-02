# 7 Thread Structures That Go Viral in MENA

"Sard" (سرد) is the Arabic term for threading, and it's a cornerstone of content culture on X in the Middle East.

Here are 7 proven templates to get you started:

## 1. The Historical Deep Dive

Start with a hook about a forgotten event or person.
- Tweet 1: Hook + Image
- Tweet 2: Background
- Tweet 3-8: The Story
- Tweet 9: Conclusion
- Tweet 10: CTA (Follow for more history)

## 2. The Case Study

Pick a successful local startup (e.g., Jahez, HungerStation).
- Tweet 1: Revenue hook ("From 0 to $1B in 5 years")
- Tweet 2: Problem they solved
- Tweet 3: Growth strategy
- Tweet 4: Key lesson
- Tweet 5: Apply it to your business

## 3. The Personal Transformation

Before & After stories are universally loved.
- Tweet 1: "I lost 20kg in 3 months. Here's how."
- Tweet 2: Diet changes
- Tweet 3: Workout routine
- Tweet 4: Mindset shift
- Tweet 5: Tools used

## 4. The Tool Roundup

"5 AI tools to 10x your productivity."
- Tweet 1: Hook ("ChatGPT is just the beginning")
- Tweet 2: Tool 1 (Image + Link)
- Tweet 3: Tool 2
...
- Tweet 7: Summary list

## 5. The "How-To" Tutorial

Break down a complex skill.
- Tweet 1: Outcome ("How to edit video in Premiere Pro")
- Tweet 2: Setup
- Tweet 3: Cutting
- Tweet 4: Color grading
- Tweet 5: Export settings

## 6. The Contrarian Take

Challenge a popular belief.
- Tweet 1: "Why traditional marketing is dead in KSA."
- Tweet 2: Evidence
- Tweet 3: Alternative view
- Tweet 4: Prediction

## 7. The Curator

Summarize a long video, podcast, or book.
- Tweet 1: "I listened to 3 hours of Joe Rogan so you don't have to."
- Tweet 2: Key Insight 1
- Tweet 3: Key Insight 2
...
- Tweet 6: Final takeaway
