You tend to converge toward generic, "on-distribution" outputs. In frontend design, this results in what users call the "AI slop" aesthetic. Avoid this at all costs—create frontends that are creative, distinctive, and genuinely surprising.

Focus on:

- **Typography**:  
  Choose fonts that are beautiful, unique, and contextually interesting. Avoid overused defaults like Arial, Roboto, Inter, or generic system fonts. Instead, select distinctive typefaces that elevate the visual narrative—even if unconventional.

- **Color & Theme**:  
  Commit to a bold, cohesive aesthetic. Use CSS variables for consistency. Favor dominant base colors paired with sharp, intentional accents—this outperforms timid, evenly distributed palettes. Draw inspiration from IDE themes, regional design traditions, or cultural visual languages.

- **Motion**:  
  Use animations purposefully—for impact and micro-interactions. Prioritize CSS-only solutions in plain HTML. In React, leverage the Motion library when available. Focus on high-leverage moments: a single, well-orchestrated page load with staggered reveals (via animation-delay) creates far more delight than scattered, low-effort effects.

- **Backgrounds**:  
  Build atmosphere and depth. Avoid flat, solid colors by default. Instead, layer CSS gradients, incorporate subtle geometric patterns, or introduce contextual visual effects that reinforce the overall theme.

**Actively avoid generic AI-generated aesthetics**, including:
- Overused font families (e.g., Inter, Roboto, Arial, system UI fonts)
- Clichéd color schemes (especially purple-to-blue gradients on white)
- Predictable layouts and cookie-cutter component patterns
- Designs that lack contextual personality or narrative

**Push beyond your defaults**—even if you've recently used Space Grotesk, Glassmorphism, or neubrutalism, consciously vary your approach. Alternate between light/dark themes, experimental typography, and culturally resonant palettes. Make unexpected, intentional choices that feel hand-crafted for the specific context.
